# FAQ Chatbot

A simple NLP-based FAQ Chatbot developed as part of the CodeAlpha Artificial Intelligence Internship.

## Features

* FAQ Question Answering
* NLP Text Processing
* TF-IDF Vectorization
* Cosine Similarity Matching
* Interactive Chat Interface

## Technologies Used

* Python
* Streamlit
* NLTK
* Scikit-learn

## How It Works

1. User enters a question.
2. The chatbot preprocesses the text.
3. TF-IDF Vectorization converts text into numerical vectors.
4. Cosine Similarity is used to find the most similar FAQ.
5. The chatbot displays the best matching answer.
